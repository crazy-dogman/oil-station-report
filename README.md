# 加油站销售数据报告生成器

## 项目简介

这是一个专为加油站设计的销售数据报告生成器，可以自动计算各项销售指标并生成标准化的日报文本。

## 功能特点

- 📊 **自动计算**：汽油/柴油吨数转换、环比变化、测评率等
- 📱 **移动端优化**：专为手机端设计，响应式布局
- 💾 **数据持久化**：自动保存输入数据，刷新页面不丢失
- 📋 **一键复制**：生成报告后可一键复制到剪贴板
- ✅ **表单验证**：智能验证输入数据，避免错误
- 🚀 **离线使用**：无需网络连接，本地即可运行

## 使用方法

1. 打开网页：[点击访问](https://您的用户名.gitee.io/oil-station-report)
2. 填写各项销售数据
3. 点击"生成报告"按钮
4. 复制生成的报告文本

## 计算公式

- **汽油吨数**：升数 × 0.00075
- **柴油吨数**：升数 × 0.00085  
- **环比变化**：(今日数据 - 昨日数据) / 昨日数据 × 100%
- **测评率**：实际评价数量 / 总消费笔数 × 100%
