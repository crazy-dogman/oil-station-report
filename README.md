## 🎯 项目简介

专为加油站员工设计的智能销售数据报告生成器，采用7步骤向导式操作流程，支持智能微信消息解析和手动数据录入，自动生成标准化的日报文本。特别针对户外阳光环境和移动端使用场景进行了深度优化。

## ✨ 核心功能

### 📊 智能数据处理
- **升转吨自动计算**：92#/95#/98#汽油、柴油精确转换
- **环比分析**：自动计算今日与昨日数据对比
- **月累计统计**：实时更新本月累计销售数据
- **智能验证**：多层数据验证，确保计算准确性

### 🤖 智能解析系统
- **微信消息解析**：一键解析昨日销售数据微信消息
- **手动数据录入**：支持9个昨日数据字段的手动输入
- **双重数据源**：智能解析 + 手动输入，确保数据完整性

### 🎨 用户体验优化
- **7步骤向导**：清晰的数据录入流程，降低操作难度
- **户外优化设计**：大字体、防眩光、高对比度界面
- **6种震撼动画**：磁悬浮、全息投影、量子粒子等60fps特效
- **智能导航**：自动滚动定位，优化长表单操作体验

### 📱 移动端专精
- **移动优先设计**：专为手机端优化的响应式布局
- **触控友好**：大按钮、合理间距，适合户外操作
- **离线可用**：单文件架构，无需网络连接

### 📋 报告生成
- **标准8部分报告**：完整的日报格式
- **一键复制**：生成后可直接复制到剪贴板

## 📖 使用指南

### 基本操作流程
1. **步骤1**：设置报告日期和加油站名称
2. **步骤2**：粘贴昨日数据微信消息或手动输入
3. **步骤3**：录入今日油品销售数据（92#/95#/98#汽油、柴油）
4. **步骤4**：录入非油销售数据
5. **步骤5**：录入电子券核销数据
6. **步骤6**：录入工行相关业务数据
7. **步骤7**：生成并复制标准报告

### 智能解析功能
- 将昨日销售数据的微信消息粘贴到步骤2
- 点击"🔍 智能解析昨日数据"按钮
- 系统自动提取11个核心数据字段
- 支持汽油、柴油、非油销售、月累计等数据

### 手动录入备选
- 当没有微信消息时，系统自动显示手动输入表单
- 支持9个昨日数据字段的精确录入
- 实时计算汇总数据，确保数据一致性

## 🔧 技术架构

### 核心技术栈
- **Alpine.js 3.14.9**：轻量级响应式框架
- **单文件架构**：所有CSS/JS内联，便于部署
- **GitHub Pages**：静态托管，全球CDN加速
- **纯前端实现**：无服务器依赖，安全可靠

### 性能优化
- **GPU硬件加速**：所有动画使用GPU渲染
- **60fps流畅动画**：专业级视觉效果
- **内存优化**：禁用localStorage，纯内存操作
- **响应式设计**：适配各种屏幕尺寸

### 计算精度
- **汽油密度**：0.75 kg/L（92#/95#/98#统一标准）
- **柴油密度**：0.85 kg/L
- **环比计算**：(今日-昨日)/昨日×100%，保留2位小数
- **升转吨公式**：升数 × 密度 ÷ 1000
